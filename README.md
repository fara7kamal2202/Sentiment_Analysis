## Project Overview  
This repository uses tools to perform **sentiment-analysis** of Amazon reviews using different methods. The aim is to extract review sentiment (positive/negative/neutral) and provide insights into customer feedback.

## Features  
- Lexicon- and rule-based sentiment analysis using NLTK (e.g., VADER).  
- Transformer-based sentiment classification (e.g., using Hugging Face models).  
- Pre-processing pipeline: cleaning review text, tokenization
- Evaluation of different methods and comparison of results.
